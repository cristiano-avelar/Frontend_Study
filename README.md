# 📚 Frontend Study

Este repositório foi criado com o objetivo de centralizar meus estudos, anotações e exercícios práticos de desenvolvimento Web Frontend. Aqui aplico conceitos fundamentais de estruturação, estilização e comportamento para consolidar meu aprendizado técnico, evitando códigos soltos e mantendo uma estrutura organizada de aprendizado.

---

## 🗂️ Organização do Repositório

O projeto é dividido em pastas específicas para cada tecnologia do ecossistema Frontend, funcionando como um guia de referência do meu aprendizado:

* **`📁 HTML/`**: Códigos focados na estruturação semântica de páginas, uso correto de tags, formulários e acessibilidade.
    * *Destaque:* `ensinamentos.html` (Guia prático com os conceitos base da linguagem).
* **`📁 CSS/`** *(Em breve)*: Exercícios de estilização, layouts responsivos (Flexbox e Grid Layout), pseudo-classes e design moderno.
* **`📁 JavaScript/`** *(Em breve)*: Scripts para lógica de programação, manipulação de DOM, eventos e consumo de APIs.

---

## 🛠️ Tecnologias e Conceitos Estudados

* **HTML5** (Estruturação semântica e boas práticas)
* **Git & GitHub** (Versionamento de código e organização de histórico com commits limpos)

---

## 🎯 Objetivos de Aprendizado

1.  Dominar a semântica do HTML5 para construir páginas bem estruturadas e acessíveis.
2.  Avançar para o CSS3 para criar interfaces visuais atraentes e totalmente responsivas.
3.  Implementar interatividade dinâmica nas páginas utilizando JavaScript nativo.
4.  Manter o repositório padronizado seguindo boas práticas de organização de arquivos do mercado.
